
**Microsoft Excel Beginner-Intermediate Test Assessment**

**Section 1: Basic Functionality**

**Q1.** What is the default file extension for an Excel workbook?

A\) .xls\
B) .xlsx\
C) .csv\
D) .pdf

**Answer:**\
B) .xlsx

**Q2.** Which of the following is the correct formula to add two numbers
in Excel?

A\) =SUM(A1 + A2)\
B) =A1 + A2\
C) =ADD(A1, A2)\
D) =SUM(A1:A2)

**Answer:**\
B) =A1 + A2

**Q3.** How do you select an entire row in Excel?

A\) Click the row number on the left side of the screen\
B) Press Ctrl + A\
C) Press Shift + Space\
D) Press Alt + Space

**Answer:**\
A) Click the row number on the left side of the screen

**Section 2: Formatting and Cell Management**

**Q4.** Which of the following is NOT a valid way to change the font
size in Excel?

A\) Use the \"Font Size\" dropdown in the Home tab\
B) Press Ctrl + Shift + P\
C) Right-click on a cell and choose \"Format Cells\"\
D) Click the \"Increase Font Size\" button in the Home tab

**Answer:**\
B) Press Ctrl + Shift + P

**Q5.** How do you merge cells in Excel?

A\) Right-click the selected cells and choose \"Merge Cells\"\
B) Click the \"Merge & Center\" button in the Home tab\
C) Press Ctrl + M\
D) Both A and B

**Answer:**\
D) Both A and B

**Q6.** What happens when you apply the \"Currency\" format to a cell in
Excel?

A\) It changes the text to bold\
B) It automatically adds the \"\$\" sign and adjusts decimal places\
C) It changes the text to italic\
D) It aligns the text to the right

**Answer:**\
B) It automatically adds the \"\$\" sign and adjusts decimal places

**Section 3: Formulas and Functions**

**Q7.** Which formula is used to calculate the sum of a range of cells
in Excel?

A\) =ADD(A1:A10)\
B) =SUM(A1:A10)\
C) =TOTAL(A1:A10)\
D) =ADDALL(A1:A10)

**Answer:**\
B) =SUM(A1:A10)

**Q8.** What is the purpose of the IF function in Excel?

A\) To add values together\
B) To check if a condition is true or false and return a result based on
that\
C) To find the average of a range of numbers\
D) To count the number of cells with text

**Answer:**\
B) To check if a condition is true or false and return a result based on
that

**Q9.** How would you create an absolute reference in a formula?

A\) Use & to connect cell references\
B) Use \$ before the column and row (e.g., \$A\$1)\
C) Use CTRL + SHIFT + A\
D) Use \* between cell references

**Answer:**\
B) Use \$ before the column and row (e.g., \$A\$1)

**Q10.** Which function would you use to calculate the average of a set
of numbers in Excel?

A\) =AVERAGE(A1:A10)\
B) =MEAN(A1:A10)\
C) =MEDIAN(A1:A10)\
D) =SUM(A1:A10) / COUNT(A1:A10)

**Answer:**\
A) =AVERAGE(A1:A10)

**Section 5: Sorting and Filtering**

**Q14.** How can you sort data in ascending order in Excel?

A\) Right-click and select \"Sort\"\
B) Click on the \"Sort & Filter\" button in the Home tab and choose
\"Sort A to Z\"\
C) Press Ctrl + A\
D) Click on the \"Insert\" tab and select \"Sort\"

**Answer:**\
B) Click on the \"Sort & Filter\" button in the Home tab and choose
\"Sort A to Z\"

**Q15.** What does the \"Filter\" feature in Excel allow you to do?

A\) Only display rows that meet certain criteria\
B) Automatically calculate sums\
C) Sort data alphabetically\
D) Format cells based on specific rules

**Answer:**\
A) Only display rows that meet certain criteria

**Section 6: Data Management and Security**

**Q16.** What is the purpose of the \"Freeze Panes\" feature in Excel?

A\) To keep the first row or column visible while scrolling through
large datasets\
B) To lock the workbook with a password\
C) To prevent other users from editing the document\
D) To automatically save the document every few minutes

**Answer:**\
A) To keep the first row or column visible while scrolling through large
datasets

**Q17.** Which of the following is the best way to protect the content
of a specific cell from being edited?

A\) Use the \"Freeze Panes\" option\
B) Password-protect the entire workbook\
C) Lock the cell by selecting \"Format Cells\" \> \"Protection\" and
then \"Protect Sheet\"\
D) Change the cell\'s font to bold

**Answer:**\
C) Lock the cell by selecting \"Format Cells\" \> \"Protection\" and
then \"Protect Sheet\"

**Q18.** To highlight cells that contain values greater than a specific
number, which tool would you use?

A\) Conditional Formatting\
B) Data Validation\
C) Text-to-Columns\
D) Pivot Tables

**Answer:**\
A) Conditional Formatting

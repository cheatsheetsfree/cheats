**ğŸ§­ Best Practices for Migration Support**

- Use **Google Workspace Migrate** or **GWMMO** for Outlook
  email/calendar migration.

- Communicate **file type differences** --- converting files to Google
  format enables real-time collaboration.

- Remind users: "Drive is your new file server."

- Encourage **Shared Drives** for team data, not personal Drives.

- Train on **labels, filters, and search** --- Gmail search syntax is
  very powerful (e.g., from:, has:attachment).

- Set company-wide **default sharing restrictions** to avoid accidental
  external sharing.

- Enable **Offline access** for Gmail, Docs, Sheets, and Slides.
